
# bigproject
A small project practice, create REST API using `Golang` + `Postgres` + `Docker` + `Redis` + `NSQ`.


## How To Run
- Install Docker
- Clone
- Run Command :
```
docker-compose up
```

## Accessing API
You can access App with BASE_URL :
```
localhost:8080
```

## Library
Library that I've used for coding
